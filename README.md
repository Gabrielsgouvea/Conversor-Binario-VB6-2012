# Conversor-Binario-VB6-2012
Conversor Binário — VB6 (2012)
Desenvolvido em Visual Basic 6 em 2012 durante o curso técnico em redes, ao estudar números binários e a função ASC do VB6. Decidi unir os dois conceitos e implementei a conversão do zero, sem referências externas.
Funcionalidades

Converte um caractere para seu valor binário usando a função ASC e divisões sucessivas por 2
Converte o binário de volta para o caractere original via Chr()

Contexto
Projeto simples mas marco pessoal — primeiro exercício de raciocínio autônomo unindo conceitos de sistema numérico binário com funções nativas da linguagem. A função de conversão desenvolvida aqui foi posteriormente reaproveitada e melhorada em outros projetos, como o encriptador em B4J.
